# 📝 Gerenciador de Tarefas - Projeto Final

Este projeto é a entrega final do **Módulo Básico** do programa **Capacita Brasil (iRede)**. O objetivo foi aplicar os conceitos fundamentais de lógica de programação e desenvolvimento em Java.

## 🚀 Sobre o Projeto
O sistema consiste em um **Gerenciador de Tarefas (To-Do List)** que permite organizar atividades do dia a dia. Através deste console application, o usuário pode cadastrar, listar e gerenciar suas pendências de forma simples e eficiente.

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Java
* **Ambiente de Desenvolvimento:** Eclipse IDE
* **Versionamento:** Git & GitHub

## 📋 Funcionalidades
* Adicionar novas tarefas.
* Listar todas as tarefas cadastradas.
* Marcar tarefas como concluídas.
* Remover tarefas da lista.
